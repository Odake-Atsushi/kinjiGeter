# kinjiGeter
近似曲線を求めるソフトウェア

# 起動
仮想環境構築
``` PowerShell
conda env create --file kinji_env.yaml
```
``` PowerShell
conda env create --file k_param_.yaml
```

環境を起動
``` PowerShell
conda activate kinjiGeter
```
``` PowerShell
conda activate kinjiGeterP
```

プログラムは，`src`の中に格納されている．
``` PowerShell
python kinjiGeter.py
```

これで起動できるはず．
